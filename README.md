# Poppins Club — сайт приватного дитячого садочка

Сучасний веб-сайт для приватного дитячого садочка з повноцінною адмін-панеллю керування контентом та інтегрованим ШІ-помічником на базі Google Gemini.

🇺🇦 Built in Ukraine · Designed & Developed by **Kovtunovych Dmytro Valeriyovych**

---

## ✨ Можливості

- 🎨 **Сучасний дизайн** — Tailwind CSS, плавні анімації, повністю адаптивний (мобілка / планшет / десктоп)
- 🛠 **Повне керування з адмінки** — всі тексти, фото, секції, картки, FAQ, групи редагуються без правки коду
- 📊 **Адмінка в стилі сайту** — теплий бірюзово-золотий теамінг, дашборд зі статистикою, робочий процес обробки заявок (статуси, нотатки, CSV-експорт)
- 🤖 **ШІ-помічник** — Google Gemini знає всю інформацію про садочок з БД (контакти, FAQ, групи, ціни)
- 📝 **Анкета попередньої реєстрації** — 6 кроків, окрема сторінка `/anketa`, валідація на кожному кроці
- 📧 **Контактна форма** — захист від спаму (honeypot + rate-limit + time-of-load + серверна валідація)
- 🖼 **Галереї фото** — карусель приміщень з drag/swipe, колаж напрямків, hero photo
- 🗺 **Google Maps** — тоlerantно парсить будь-який формат (повний iframe, markdown, URL)
- 🔐 **GDPR-чекбокс** — згода на обробку персональних даних на всіх формах

## 🚀 Локальний запуск

### Передумови
- Python 3.11+
- Node.js 20+
- npm

### 1. Клонувати репозиторій
```bash
git clone https://github.com/Ke1fosao/Poppins_Club.git
cd Poppins_Club
```

### 2. Backend (Django)
```bash
cd backend

# Створити virtualenv
python -m venv ../.venv
# Windows:
..\.venv\Scripts\activate
# Linux/Mac:
source ../.venv/bin/activate

# Встановити залежності
pip install -r requirements.txt

# Застосувати міграції (вже містять початкові дані: секції, картки, групи, FAQ)
python manage.py migrate

# Створити суперюзера для входу в /admin/
python manage.py createsuperuser

# Запустити dev-сервер
python manage.py runserver
```
Бекенд буде на `http://127.0.0.1:8000/`, адмінка — `http://127.0.0.1:8000/admin/`.

### 3. Frontend (React + Vite)
У новому терміналі:
```bash
cd frontend
npm install

# Скопіюйте .env.example → .env та заповніть VITE_GEMINI_API_KEY
cp .env.example .env
# або вручну скопіювати у Windows

npm run dev
```
Фронтенд буде на `http://localhost:5173/`.

## 🎛 Адмін-панель

Адмінка стилізована під сайт (тепла бірюзово-золота палітра) і має **дашборд** на
головній: лічильники нових заявок/повідомлень, останні заявки, швидкі переходи.

Моделі згруповано у **3 логічні секції** бічного меню (а не одним блоком «API»),
у фіксованому, зрозумілому порядку:

**📥 Звернення від батьків** *(щоденна робота — зверху)*
| Розділ | Що там |
|---|---|
| **Заявки з анкети** | Усі заявки з 6-крокової анкети. Робочий процес: статус, «прочитано», нотатки, масові дії, **експорт у CSV (Excel)** |
| **Повідомлення з форми** | Запити з контактної форми з тим самим робочим процесом і CSV-експортом |

**⚙️ Налаштування та тексти** *(клік веде одразу у форму — без проміжного списку)*
| Розділ | Що там |
|---|---|
| **Контакти та соцмережі** | Телефон (з валідацією формату), email, адреса, карта, FB/IG/Telegram/Threads, логотипи, текст у шапці й футері |
| **Тексти розділів** | Всі заголовки/описи кожної секції, hero-фото, бейдж досвіду, тези про садочок |

**🏠 Секції головної сторінки**
| Розділ | Що там |
|---|---|
| **«Про нас» — картки переваг** | Переваги (заголовок, опис, іконка, колір) |
| **Напрямки: картки-тези** | Картки, розподілені на 2 ряди (картка + опис + іконка + колір) |
| **Напрямки: фото каруселі (ряд 1)** | Фото 16:9 з порядком |
| **Напрямки: фото колажу (ряд 2)** | 3 фіксовані позиції з конкретними аспектами |
| **Приміщення — слайди** | Карусель приміщень (фото + підпис) |
| **Вікові групи** | Назва, вік, опис, особливості (списком), іконка, «найпопулярніша» |
| **Питання та відповіді (FAQ)** | FAQ з порядком |

### 🗂 Робочий процес обробки звернень

Заявки й повідомлення — не «звалище даних», а керований список:

- **Статус** — `🆕 Нова → 📞 В роботі → ✅ Зв'язались → 🎉 Зараховано / 🚫 Відмова / 📦 Архів`.
  Можна міняти прямо у списку або масово для кількох записів.
- **Переглянуто** — нові заявки позначені бірюзовою крапкою; автоматично гасне при відкритті.
- **Нотатки адміністратора** — внутрішні нотатки для команди (відвідувач їх не бачить).
- **CSV-експорт** — вивантаження вибраних заявок у Excel-сумісний CSV (з BOM, кирилиця ціла).

> **Структура даних.** Мультивибір анкети (вік, формат, дні, час, очікування) зберігається
> коректними JSON-списками, а відповіді «Так/Ні» — булевими полями (а не текстом), тож дані
> придатні для фільтрації, аналітики й експорту.
